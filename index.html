About Me

I'm a former bio-research tech, turned code cruncher and bug slayer. My first programming languages were HTML and CSS when I was thirteen. I wanted to design a pretty page for my Neopets guild, so I taught myself what I needed to know. I enjoy solving problems from a logical perspective. Projects

  • Babeling Book

    live | github
    A single-page blogging site with posts and comments


    The site is for people to practice writing in a foreign language about the things they care about and get helpful feedback from members of the community so they can improve their grammar and spelling. Currently, users can create an account and log in to make posts and leave comments.

  • PokeProbe

    live | github
    Product Hunt clone where users catch and make profiles for the Pokemon they catch

    Users can catch wild, randomly generated Pokemon fusions. If they don't like the Pokemon they first see, they can run away and try again. They can then name these pokemon and create profiles for them that include a description and the element types the Pokemon is made of.

  • SpudHub

    live | github
    A single-page Etsy clone to buy and sell overpriced produce

    Users/stores to can add produce they want to sell. Users can also add and remove products from their shopping cart. The shopping cart persists even if the user logs out and in again, or if the user refreshes the page.
